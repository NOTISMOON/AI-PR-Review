# AI Code Review Tool

This project is a Next.js app for analyzing GitHub pull requests with AI-assisted review insights. The original design is available at https://www.figma.com/design/XXxoL6MioJORlXuthdDndS/AI-Code-Review-Tool.

## Running the code

Install dependencies:

```bash
pnpm install
```

Start the development server:

```bash
pnpm dev
```

Build for production:

```bash
pnpm build
```
